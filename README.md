# Spendora-Personal_Finance_Analytics_Deshboard
Personal Finance Analytics Dashboard built with Python and Streamlit for tracking income, expenses, savings, and financial risk insights.

Key Features:-
 - Secure Authentication
 - Income & Expense Management
 - Financial Analytics Dashboard
 - Interactive Data Visualization
 - Automated Financial Reports

TECH STACKS:-
Programming Language:-
-  Python
  
Framework:-
- Streamlit

Libraries:-
- Pandas – Data analysis and processing
- NumPy – Numerical computations
- Plotly – Interactive visualizations
- Matplotlib – Data visualization and charts

Database
- SQLite – Lightweight relational database

Additional Functionalities:-

- Email services for OTP verification
- Automated PDF report generation.
